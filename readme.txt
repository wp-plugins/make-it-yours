Gives you two panels to insert custom CSS and PHP codes, so you can customize your theme.

== Description ==

Without hurting your template files, just put your custom CSS and PHP codes and enjoy the magic. It worth to mention that PHP codes will run safely, so your site won't break if you misspell or typed wrong codes.

== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Appearance -> Custome CSS or Custom Functions pages
5. Enjoy!
